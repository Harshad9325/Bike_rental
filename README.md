# bike-rental

React + Vite + Tailwind CSS + Framer Motion demo project (frontend-only)

Theme: Blue & White  
Style: Colorful + Premium animations  
Navbar: Fixed at the top

## Quick start

1. Unzip the project.
2. From the project root run:

```bash
npm install
npm run dev
```

3. Open the local dev server (usually http://localhost:5173)

This is a frontend-only demo. Bike images are loaded from Unsplash demo URLs. Replace the images in `src/components/Bikes.jsx` with local assets if you prefer.
